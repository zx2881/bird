# 任务：构建“全球鸟类分布可视化知识图谱”网页应用

## 1. 目标
请协助我从零开始搭建一个网页，能够通过交互式图谱和地图展示鸟类分布。要求不仅提供代码，还要提供完整的网页部署流程说明。

## 2. 网页架构需求
### 【前端：交互界面】
- 框架 Vue 3 + Vite
- UI 风格 现代、简洁。使用 Element Plus 组件库。
- 页面布局 (Layout)
    - Header 标题“全球鸟类多样性知识探索平台”。
    - Main Area 
        - 左侧 30%：搜索框 + 实体详情展示（显示鸟类习性、保护等级等）。
        - 右侧 70%：上方为 ECharts 力导向图（Knowledge Graph），下方为 Leaflet 地图。
- 交互逻辑
    - 点击图谱中的【鸟类节点】，左侧显示其详细信息。
    - 点击图谱中的【地点节点】，下方地图自动跳转并标记该坐标。

### 【数据：后端静态驱动】
- 模式 采用 `static-data` 模式。
- 要求 请帮我生成一份包含 10 种代表性鸟类（如：丹顶鹤、朱鹮、中华秋沙鸭、勺嘴鹬等）的模拟 `knowledge.json` 文件。
- 数据结构 包含 `nodes` 和 `links`，且 `nodes` 中必须附带经纬度属性。

### 【自动化：数据爬取脚本】
- 请提供一个 Python 脚本，演示如何使用 `pygbif` 抓取数据并自动生成上述网页所需的 `knowledge.json`。

## 3. 分步指导要求
请按以下顺序给出代码：
1. 项目初始化指令：如何使用 npm 创建项目并安装 ECharts, Leaflet, Element Plus。
2. 核心代码实现：给出 `App.vue` 的完整代码。
3. 样式美化：给出必要的 CSS。
4. 运行说明：如何在本地预览网页，以及如何将其打包成可以放在服务器上的静态 HTML 文件夹。
